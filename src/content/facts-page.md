# Product

**Product category:**  
Online whiteboard software

**Additional categories:**

- Real-time collaboration software
    
- Agile project management software
    
- Retrospective software

**Product description:**  
Ludi is an online whiteboard used by teams to run planning sessions, retrospectives, brainstorming sessions, and collaborative workshops in a shared visual workspace.

---

# Core Use Cases

Teams use Ludi for collaborative sessions such as:

- Sprint planning
    
- Backlog refinement
    
- Retrospectives
    
- Brainstorming sessions
    
- Workshops
    
- Project kickoffs
    

These sessions are conducted on a shared visual board that enables real-time collaboration between participants.

---

# Key Product Capabilities

## Collaborative Whiteboard

- Infinite collaborative canvas
    
- Sticky notes, shapes, arrows, and images
    
- Live cursors for real-time collaboration
    

These features enable teams to express ideas visually and collaborate on a shared board.

---

## Facilitation Tools

Ludi includes features designed for running structured meetings:

- Voting rounds
    
- Timers
    
- Facilitation controls
    
- Activity frames
    
- Participant navigation across the board
    

These tools help guide participants through collaborative sessions.

---

## Templates

Ludi provides a template library for meetings and workshops.

Examples include:

- Sailboat retrospective
    
- Start / Stop / Continue retrospective
    
- Lean Canvas
    
- Story Mapping
    
- MoSCoW prioritization
    
- Crazy 8's brainstorming
    

Templates can be customized or combined to create boards for specific workflows.

---

## Engagement Features

Ludi includes interactive tools designed to encourage participation in meetings:

- Icebreaker activities
    
- Confetti effects
    
- Interactive elements for workshops
    

These features are intended to make remote sessions more engaging for participants.

---

## Action Tracking

Boards can include action items derived from meeting notes.

Features include:

- Assigning owners to actions
    
- Setting due dates
    
- Reminder notifications

# Security

Ludi implements multiple security controls.

## Application Security

- TLS 1.2 encryption for connections
    
- AES-256 encryption for data at rest
    
- AES-256 encrypted backups
    
- Backups retained for up to 7 days
    

## Operational Security

- Multi-factor authentication for system accounts
    
- VPN-restricted network access
    
- Role-based access control
    
- Least-privilege access principles
    

## Authentication

Users can sign in with:

- Email and password
    
- Google
    
- GitHub
    
- Slack
    
- LinkedIn
    

SAML-based single sign-on is available on Business and Enterprise plans.
